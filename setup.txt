Sigere – Guía de configuración rápida
===================================

1. Requisitos previos
   • Python 3.11 instalado (macOS: brew install python@3.11 , Windows: winget install Python.Python.3.11).  
   • Git (opcional, para clonar el repositorio).
  

2. Clonar u obtener el proyecto
   • Desde Git: `git clone https://github.com/MiguelVN7/Proyecto-Integrador-1`  
   • O copia el contenido del proyecto en una carpeta local.  
   
3. Crear entorno virtual (recomendado)
   • Windows (PowerShell):
     ```powershell
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
     ```
   • macOS/Linux (bash/zsh):
     ```bash
     python3 -m venv .venv
     source .venv/bin/activate
     ```

4. Instalar dependencias
   • Con el entorno virtual activo:
     ```
     pip install -r requirements.txt
     ```

5. Variables de entorno (.env)
   • El proyecto usa `python-dotenv` y espera un token para la integración con OpenAI.  
   • Crea un archivo `.env` en la raíz con al menos:  
     ```
     OPENAI_API_KEY=dummy
     ```
   • Si no se cuenta con una clave válida, se puede dejar `dummy` para desarrollo local.  

6. Migraciones de base de datos
   • Ejecutar las migraciones para crear las tablas de SQLite (incluye el modelo `PerfilUniversitario` y reseñas):
     ```
     python manage.py migrate
     ```
   • (Opcional) Crear un usuario administrador para acceder al panel de Django:
     ```
     python manage.py createsuperuser
     ```
   • (Opcional) Cargar datos de ejemplo si se desea poblar la base con reservas iniciales:
     ```
     python populate.py
     ```

7. Levantar el servidor de desarrollo
   • Asegurarse de mantener el entorno virtual activo y el `.env` configurado.  
   • Ejecutar:
     ```
     python manage.py runserver
     ```
   • Si se usa PowerShell y necesitas definir la variable temporalmente:  
     ```powershell
     $env:OPENAI_API_KEY="dummy"; python manage.py runserver
     ```

     Yo (Miguel) lo corro así en mi windows:
     ```powershell
     $env:DJANGO_SETTINGS_MODULE="Sigere.settings"; $env:OPENAI_API_KEY="dummy"; C:/Users/Miguel/AppData/Local/Microsoft/WindowsApps/python3.11.exe manage.py runserver
     ```

8. Rutas principales de la aplicación
   • Página de aterrizaje: http://127.0.0.1:8000/  
   • Inicio (home): http://127.0.0.1:8000/home/  
   • Inicio de sesión: http://127.0.0.1:8000/login/  
   • Panel de reservas: http://127.0.0.1:8000/misReservas/  
   • Panel de reseñas: http://127.0.0.1:8000/resenas/
   • Perfil de usuario: http://127.0.0.1:8000/perfil/  

9. Comprobaciones recomendadas
   • Verificar que todo esté correcto antes de ejecutar el servidor:
     ```
     python manage.py check
     ```
   • Si se modifica el modelo, volver a ejecutar `makemigrations` y `migrate`.
